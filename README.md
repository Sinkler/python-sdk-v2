# Cardpay APIv3 Python SDK

You can sign up for a Cardpay account at https://www.cardpay.com.

## Getting Started

Please follow the [installation](#installation) instruction and take a look at [usage examples](tests).


## Requirements

Python 2.7 and 3.4+

## Installation & Usage
### pip install

If the python package is hosted on Github, you can install directly from Github

```sh
pip install git+https://github.com/cardpay/python-sdk-v3.git
```
(you may need to run `pip` with root permission: `sudo pip install git+https://github.com/cardpay/python-sdk-v3.git`)

Then import the package:
```python
import cardpay 
```

### Setuptools

Install via [Setuptools](http://pypi.python.org/pypi/setuptools).

```sh
python setup.py install --user
```
(or `sudo python setup.py install` to install the package for all users)

Then import the package:
```python
import cardpay
```
